# Mi Hogar Home Cleaning - Landing Page

## Project Overview
Landing page for "Mi Hogar Home Cleaning" - a bilingual home cleaning service in Bow Valley, Canmore, Canada.

## Architecture
- **Frontend**: React.js with Tailwind CSS
- **Backend**: FastAPI (basic setup, not used for this landing page)
- **Database**: MongoDB (not used for this landing page)

## User Personas
1. **Local Homeowners** - Residents in Bow Valley/Canmore area needing cleaning services
2. **Vacation Rental Owners** - Property managers needing specialized cleaning
3. **Spanish-speaking clients** - Bilingual service customers

## Core Requirements (Static)
- [x] Responsive navigation menu
- [x] Hero section with mountain view imagery
- [x] Services section with custom icons (5 services)
- [x] Contact footer with map, phone, email
- [x] Bilingual text support (English/Spanish)
- [x] Mobile-responsive design
